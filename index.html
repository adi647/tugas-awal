Tugas Bootstrap

Apa itu Bootstrap & Kenapa Dipakai?

Bootstrap adalah framework front-end berbasis HTML, CSS, dan JavaScript yang digunakan untuk mempermudah pembuatan website yang responsif dan mobile-first.

Kenapa Bootstrap banyak dipakai?

  • Mempercepat pembuatan website (banyak komponen siap pakai).
  • Desain otomatis responsif (bisa menyesuaikan dari HP sampai layar besar).
  • Tampilan modern tanpa perlu coding CSS dari nol.

Cara Install Bootstrap

1. Via CDN (lebih cepat & praktis)

Tambahkan kode berikut di dalam tag <head> HTML:

<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet">

2. Via Lokal (offline)

- Download file Bootstrap dari getbootstrap.com
- Simpan di folder project (misal css/bootstrap.min.css)
- Tambahkan link ke HTML: 

<link href="css/bootstrap.min.css" rel="stylesheet">

Struktur Dasar HTML + Bootstrap

Setiap file HTML Bootstrap biasanya terdiri dari <head> (berisi link CSS Bootstrap) dan <body> (konten website). 

<!DOCTYPE html>
<html>
  <head>
    <link href="bootstrap.min.css" rel="stylesheet">
  </head>
  <body>
    <div class="container"> ... </div>
  </body>
</html>
